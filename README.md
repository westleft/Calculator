設計稿由六角學院所提供，詳細請參考 [新手 JS 地下城](https://courses.hexschool.com/p/javascript-js)。

為了練習 TypeScript 將原本的原生 JS 重構，並修掉了部分 BUG，例如：

* 運算符重複  `200+-200`
* 多個小數點  `0.11.22`
* 結尾非數字  `200+200+` or `200-200.`
* 小數點位置  `.200`

功能大致正常，但可能有意想不到的 BUG 還沒解決。
